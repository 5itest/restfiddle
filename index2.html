Request Builder

Make requests with ease. Header, query param, auth and much more supported. Organize requests in the form of a tree structure inside different projects in a workspace.

Collaboration

Free yourself from the hassles of sharing APIs via chats or emails. Adding collaborators for a workspace does all the magic for you. Sharing different APIs with different sets of people has never been so easy!

Environment

Run same set of APIs for different environments with ease. Use expression language in API urls. Environment-specific values of variables get populated while running the APIs.

Assert

Get rid of manually comparing response of requests, every time you run them. Save Asserts to do the work for you. When a request is run, all the Assert results get evaluated against expected values.

Report

Instead of running requests individually, run requests in a project or folder and get a PDF report of the results.

Backend

Quick entity creation, user and role management services are provided through APIs to all app developers to help them develop their apps at a faster pace and without much effort.
